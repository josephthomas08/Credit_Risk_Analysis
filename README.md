# Credit_Risk_Analysis
Predict credit risk 

## PROJECT OVERVIEW

The Credit risk analysis could be very complex in fast changing world with possible impact from many possible factors. And credit risk prediction has always been evolving with the market needs for fast resolution and decision making process. Data Scientist have been using Machine learning to assist us to create multpile models , evlauate the models and eventually train the models.

In This project we using imbalanced -learn and scikit-learn models to evaluate them using a resampling method . The plan i want pursue in the project is use the oversampling, undersampling, combinational approach along with two new machine learning models that will reduce the bias to predict the possible credit risk.
